# 🖋️ 飞白 (Feibai) 

**东方暗黑水墨风 · 卡牌肉鸽 (Roguelike) RPG**
> 本项目为 GameJam 参赛作品，完全基于原生 Web 技术（HTML5 / Canvas / Vanilla JS）开发，无需任何插件或引擎即可在浏览器中流畅体验泼墨挥毫的武侠世界。

## 📜 游戏简介 (Game Pitch)

《飞白》是一款融合了中国传统水墨画风与硬核卡牌构筑（Deck-building）玩法的轻度 RPG 游戏。玩家将坠入幽冥地府，在“望乡台”与“奈何桥”之间游历，面对冥府的游魂与恶鬼。

游戏将书法中的“飞白”笔意与战斗动作深度结合。玩家在拖拽卡牌时不仅能留下真实的水墨残影，还要在出牌时兼顾“平仄”韵律与“气势”的积攒，体验刀光剑影与诗词歌赋交织的独特国风浪漫。

## ✨ 核心特色 (Core Features)

* 🗡️ **泼墨挥毫的战斗交互**
  利用底层 Canvas 粒子系统，玩家在拖拽卡牌时会产生动态的“飞白”墨迹残影。卡牌生效时伴有 Hit-Stop（定格震动）与书法字体伤害飘字，带来极强的打击反馈。
* ☯️ **“气”与“势”的双轨博弈**
  * **气 (Energy)**：传统的出牌费用限制。
  * **势 (Momentum)**：连击积攒系统。不断打出攻击卡牌积攒“势”，满 10 点后下一次攻击伤害提升 50%，一击破阵！
* 📜 **独创“平仄”与被动“闪避”系统**
  每张卡牌带有“平”或“仄”的属性，打出后会在空中留下诗词残卷的视觉印记；独特的“闪避”卡牌无法主动打出，而是作为护身符在敌方攻击时自动弃置抵消伤害。
* 🗺️ **六道轮回与冥府探索**
  多流派选择（剑、弓、枪、毒、棍、拳），结合分支地图探索（山路战斗、茶楼奇遇、破庙抉择），每一次轮回都是全新的故事。

## 🛠️ 技术栈 (Tech Stack)

本项目为了保证极致的跨平台兼容性与部署便捷性，**零外部依赖**：
* **结构与样式**：`HTML5` + `CSS3` (包含复杂的 3D 浮雕 UI、过渡动画与阴影滤镜)。
* **核心逻辑**：`Vanilla JavaScript` (原生 JS 实现状态机、杀戮尖塔式的卡牌抽弃逻辑、敌人 AI)。
* **视觉特效**：`Canvas API` (实现背景水墨粒子游动、鼠标轨迹渲染)。

## 🚀 运行指南 (How to Run)
### 本地游玩：

克隆或下载本仓库到本地。

确保所有图片和音频文件均已正确放置在 assets/ 目录下。

双击运行 index.html 即可在任意现代浏览器（Chrome/Edge/Firefox）中开始游戏。
(注：由于浏览器的自动播放策略，本地直接双击打开可能导致开屏 BGM 无法自动播放，建议通过 Live Server 等本地服务器运行)

### 线上部署：
本项目已通过 GitHub Pages 部署，任何人都可以通过外网链接直接访问游玩。
